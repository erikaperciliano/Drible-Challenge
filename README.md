# Drible Project

This project is a challenge to Dribble 

### 📋 Prerequisites

```
VS Code
```
### 🔧 Installation

```
git clone https://github.com/erikaperciliano/Drible-Challenge.git
npm i 
```
### 📦 In the project directory, you can run:

```
npm run dev
```
